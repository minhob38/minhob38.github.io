# Git

git remote update : 원격 브랜치들을 가져와, 로컬 브랜치들을 업데이트 합니다.  
git branch -r : 원격 브랜치들을 볼 수 있습니다.  
git branch -a : 로컬 브랜치들을 볼 수 있습니다.  
git chekout -t [remote repository]: 원격저장소를 가져올 수 있음
